Openclaw 係一個好用嘅工具，以下係我平時常用嘅 commands，方便大家參考:

## Basic Commands

```bash
# 查看版本
openclaw --version

# 查看幫助
openclaw --help
```

## Common Operations

```bash
# 去設定 openclaw
openclaw onboard

# restart gateway
openclaw restart gateway

```

## Tips

記得經常使用 `--help` 參數嚟查看可用嘅選項同參數。

Hope you find it useful
